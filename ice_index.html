Prophet Card

This was an introductory activity piecing togther both HTML and CSS using a wireframe template. The goal was to take a provided template and match it by creating a wireframe to list our HTML elements and designing it using CSS styles.

Apostle Cards

The goal of this exercise was to practice using divs and grid structures to place images on a page. A secondary focus was to work on getting a proper background image into the page.

Favorite Devo

This activity introduce table and form HTML elements. As of right now, forms have no real function but, with Javascript, we can correct that in future classes.

Holiday Page

In this activity, each student was given the same HTML but we told to write individual CSS. the purpose was to show how different frames can be deisgned differently from different viewpoints.

Grid Flags

The focus for this activity was to use grids to create the lines and shapes on each flag. In the CSS we created each stripe by stretching the blocks over multiple columns and rows. For the stars, there was a specific element called "clip-path".

Positioning

Similar to the previous activity, the focus was on positioning HTML items in specific places on the page. Each part had a template to follow and, using CSS, it was our task to match our page to the template.

File Size

The point of this exercise is to show how file sizes can effect the load times for a page. In order to speed up load times, we took each image (png) from the gallery and changed file type to .jpg and, for the thumbnails, saved it as a new file with scaled down dimensions.
